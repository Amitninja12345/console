Un grupo de servidores es una colección de instancias de servidor que se configuran y administran como una sola. En un dominio administrado, toda instancia del servidor de aplicaciones pertenece a un grupo de servidores incluso si es el único miembro. Las instancias del servidor en un grupo comparten la misma configuración de perfil y contenido implementado.

Un controlador de dominios y un controlador host refuerzan la configuración estándar en todas las instancias del servidor para todo grupo de servidores en su dominio.

Un dominio puede consistir de múltiples grupos de servidores. Diferentes grupos de servidores se pueden configurar con diferentes perfiles e implementaciones. Por ejemplo, un dominio se puede configurar con diferentes capas de servidores proporcionando diferentes servicios.

Diferentes grupos de servidores también pueden tener el mismo perfil e implementaciones. Por ejemplo, esto puede permitir el realizar actualizaciones de aplicaciones en donde la aplicación se actualice en un grupo de servidores y luego se actualice en un segundo grupo de servidores, evitando un corte completo del servicio.
