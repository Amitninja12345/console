elytron サブシステムの以下のリソースにランタイム操作のセットを提供します。

  • クレデンシャルストア (Credential Store): エイリアスを追加、読み取り、および削除し、エイリアスのパスワードを設定します。また、クレデンシャルストアをリロードします。
  • フィルタリングキーストア (Filtering Key Store): エイリアスを読み取りおよび削除します。
  • キーストア (Key Store): エイリアスを読み取り、変更、および削除します。証明書をインポートおよびエクスポートします。キーペアを生成します。証明書署名リクエストを生成します。キーストアをロードおよび保存します。
  • LDA_ キーストア (LDAP Key Store): エイリアスを読み取りおよび削除します。
