Visualiser les journaux du serveur et des applications afin d'aider à diagnostiquer les erreurs, les problèmes de performance et autres problèmes. Pour qu'un journal soit visible, il doit être situé dans le répertoire jboss.server.log.log.dir du serveur. La console respecte également l'attribution des rôles RBAC de l'utilisateur, de sorte qu'un utilisateur ne peut voir que les journaux auxquels il peut accéder.

Par défaut, la console affiche les 2000 dernières lignes d'un fichier journal. Si vous voulez voir le fichier journal dans son ensemble, veuillez sélectionner «Télécharger ». Pour visualiser ou suivre le fichier journal dans une fenêtre de navigateur séparée, sélectionner « Ouvrir »dans la fenêtre externe.
