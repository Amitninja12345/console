A deployment represents anything that can be deployed (e.g. an application such as EJB-JAR, WAR, EAR, any kind of standard archive such as RAR or JBoss-specific deployment) into a server.

You can use drag and drop to add new content or replace existing deployments. Simply drag one or several files onto the column. If there's already a deployment with the same name, the deployment will be replaced, otherwise the deployment will be added.
