Bean Validation est une spécification Java qui

  • vous permet d'exprimer des contraintes sur les modèles d'objets par le biais d'annotations
  • vous permet d'écrire des contraintes personnalisées de manière extensible
  • fournit des API pour valider les objets et les graphiques d'objets
  • fournit des API pour valider les paramètres et retourne des valeurs de retour de méthodes et de constructeurs
  • signale l'ensemble des violations (localisées)
  • fonctionne en Java SE, mais est intégré dans Java EE (6 et 7 ; Bean Validation 2.0 doit bientôt faire partie de Java EE 8)

build.fullName} utilise Hibernate Validator pour son implémentation par lots.
