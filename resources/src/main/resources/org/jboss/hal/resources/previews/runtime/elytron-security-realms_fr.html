Fournit un ensemble d'opérations d'exécution pour les ressources suivantes du sous-système elytron :

  • Caching Realm : Supprime toutes les entrées du cache.
  • Domaine personnalisable et modifiable sur mesure : Ajoute, lit et supprime des identités. Définit également un mot de passe pour une identité.
  • Filesystem Realm: Ajoute, lit et supprime des identités. Définit également un mot de passe pour une identité.
  • LDAP Realm : Ajoute, lit et supprime des identités. Définit également un mot de passe pour une identité.
  • Properties Realm: Recharge les fichiers de propriétés à partir du système de fichiers.
