Fournit un ensemble d'opérations d'exécution pour les ressources suivantes du sous-système elytron :

  • Compte d'autorité de certification : Crée, désactive et met à jour un compte et modifier la clé de compte.
  • Gestionnaire de clés : Réinitialise le gestionnaire clé.
  • Domaine de sécurité : Lit une identité.
  • Gestionnaire de certificats de confiance : Recharge la liste de révocation des certificats.
