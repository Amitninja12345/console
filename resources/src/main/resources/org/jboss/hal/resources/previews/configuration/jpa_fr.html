Gère les exigences gérées par conteneur de l'API de persistance Java (JPA) 2.1 et vous permet de déployer des définitions d'unités persistantes, des annotations et des descripteurs.

L'API de persistance Java (JPA) est une spécification Java pour l'accès, la persistance et la gestion des données entre les objets ou classes Java et une base de données relationnelle. La spécification JPA reconnaît l'intérêt et le succès de l'objet transparent ou du paradigme de mappage relationnelle. Il standardise les API de base et les métadonnées nécessaires à tout objet ou mécanisme de persistance relationnel.
