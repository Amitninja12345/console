Le sous-système elytron est nouveau dans ${build.fullName}. Il est basé sur le projet WildFly Elytron, qui est un framework de sécurité utilisé pour unifier la sécurité sur l'ensemble du serveur d'application. Le sous-système elytron permet un point de configuration unique pour la sécurisation des applications et des interfaces de gestion. WildFly Elytron fournit également un ensemble d'API et de SPI pour fournir des implémentations personnalisées de fonctionnalités et l'intégration avec le sous-système.

Cet affichage de runtime fournit des informations dynamiques sur les différents magasins, domaines de sécurité et ressources SSL qui sont configurés pour ce serveur.
