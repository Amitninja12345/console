elytron サブシステムの以下のリソースにランタイム操作のセットを提供します。

  • キャッシングレルム (Cashing Realm): キャッシュからすべてのエントリーを削除します。
  • カスタム編集可能レルム (Custom Modifiable Realm): アイデンティティーを追加、読み取り、および削除します。アイデンティティーのパスワードも設定します。
  • ファイルシステムレルム (Filesystem Realm) : アイデンティティーを追加、読み取り、および削除します。アイデンティティーのパスワードも設定します。
  • LDAP レルム (LDAP Realm): アイデンティティーを追加、読み取り、および削除します。アイデンティティーのパスワードも設定します。
  • プロパティーレルム (Properties Realm): ファイルシステムからプロパティーファイルをリロードします。
