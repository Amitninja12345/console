Un pool de mémoire tampon d'octets utilisé pour les opérations d'E/S, qui offre les mêmes capacités que le pool de mémoire tampon du sous-système E/S, pouvant ainsi être utilisées de manière interchangeable et doivent avoir un nom unique. Ce pool de mémoire tampon permet une configuration plus précise de la quantité totale de mémoire conservée que le sous-système E/S.
