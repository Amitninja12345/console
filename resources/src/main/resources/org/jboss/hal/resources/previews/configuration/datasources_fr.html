Les deux types généraux de ressources sont appelés sources de données et sources de données XA.

  • Les sources de données non-XA sont utilisées pour des applications qui n'utilisent pas de transactions, ou des applications qui utilisent des transactions avec une seule base de données.
  • Les sources de données XA sont utilisées par des applications dont les transactions sont réparties sur plusieurs bases de données. Les sources de données XA introduisent des frais généraux supplémentaires.
