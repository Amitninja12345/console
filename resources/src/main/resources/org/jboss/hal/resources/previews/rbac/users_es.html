What a management user is permitted to do is determined by the roles to which the user is assigned. A system of includes and excludes based on the user membership determines to which role a user belongs.

Se considera que un usuario está asignado a un rol si:

  1. El usuario:
    • está enumerado como un usuario que se debe incluir en el rol; o
    • es un miembro de un grupo que está enumerado para incluirse en el rol.
  2. El usuario no:
    • está enumerado como un usuario que se debe excluir del rol; o
    • es un miembro de un grupo que está enumerado para excluirse del rol.

Las exclusiones tienen prioridad sobre las inclusiones.
