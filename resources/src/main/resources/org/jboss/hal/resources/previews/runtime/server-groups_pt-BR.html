Um grupo de servidor é uma coleção de instâncias de servidor que são gerenciadas e configuradas como uma única. No managed domain, cada instância do servidor do aplicativo pertence a um grupo do servidor, mesmo quando este é o único membro. As instâncias do servidor num grupo compartilham a mesma configuração de perfil e conteúdo implantado.

Um domain controller e um host controller reforçam a configuração padrão em todas as instâncias do servidor de cada grupo do servidor em seu próprio domain.

Um domain pode consistir de diversos grupos de servidor. Grupos diferentes de servidores podem ser configurados com perfis e implantações diferentes. Um domain pode ser configurado com camadas de servidores diferentes fornecendo serviços diferentes, por exemplo.

Grupos diferentes de servidores podem também possuir o mesmo perfil e implantações. Isto pode, por exemplo, permitir atualizações do aplicativo progressivas onde o aplicativo é atualizado em um grupo do servidor e então atualizado num segundo grupo de servidor, evitando uma interrupção do serviço completa.
