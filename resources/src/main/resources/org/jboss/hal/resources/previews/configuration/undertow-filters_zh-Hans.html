过滤器可以修改请求的某些方面，且可以使用 predicate 来控制过滤器何时执行。过滤器的常见用例包括：设置头数据、进行 GZIP 压缩。

您可以定义下列过滤器类型：

  1. Custom Filter
  2. Error Page
  3. Expression Filter
  4. GZip
  5. Mod Cluster
  6. Request Limit
  7. Response Header
  8. Rewrite

Note
A filter is functionally equivalent to a global valve used in previous versions of ${build.shortName}.
