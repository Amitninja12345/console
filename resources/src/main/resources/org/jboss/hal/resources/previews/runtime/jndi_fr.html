Fournit une vue d'ensemble de l'espace noms JNDI local. La spécification de la plate-forme Java EE définit les contextes JNDI suivants :

  • java:comp - L'espace de nommage est étendu au composant actuel (EJB)
  • java:module - Limité au module en cours
  • java:app - Limité à l’application en cours
  • java:global - Limité au serveur d'application

En plus des espaces de noms standard, ${build.shortName} fournit également les deux espaces de noms globaux suivants :

  • java:jboss
  • java:/ / java

Veuillez noter que seules les entrées du contexte java:jboss/exportés sont accessibles via JNDI distant. Pour les déploiements web, java:comp est aliasé en java:module, de sorte que les EJB déployés dans un war n'ont pas leur propre espace de noms comp.
